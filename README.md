# 🐾 CRUD de Animais com Flask

Este é um projeto simples de CRUD (Create, Read, Update, Delete) desenvolvido com Python e Flask.

O sistema permite cadastrar, visualizar, editar e excluir animais diretamente pelo navegador.

---

## 🚀 Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML (com Jinja)

---

## 📚 Funcionalidades

- ➕ Adicionar novos animais  
- 📋 Listar todos os animais  
- ✏️ Atualizar informações  
- ❌ Deletar registros  

---

## 🧠 Como funciona

O sistema utiliza:

- Banco de dados SQLite para armazenar os dados  
- Flask para gerenciar rotas e lógica  
- Templates HTML para exibir os dados no navegador
